Обо мне

Привет! Меня зовут Максим Истляев, я из города Саранск (Россия). Работаю в области фронтэнда: html, css, javascript. В данный момент ищу постоянную удалённую работу с окладом от 65 000 рублей.

Основные знания

  • html и css. Хорошо знаю html и css, умею делать семантическую и валидную верстку. Проверяю pixel perfect.
  • html5 и css3. Вендорные префиксы, progressive enhacement, graceful degradation и т.д.
  • javascript и jQuery. Хорошее знание jQuery, чистый javascript знаю средне — мало практики, так как он нечасто используется в проектах.
  • backbone.js. Средний уровень знания, имеется реализованный проект.
  • Шаблонизаторы smarty, template toolkit. Также имеется опыт интеграции вёрстки в WordPress, phpBB, Slaed, LiveJournal, NetCat.
  • responsive, adaptive. Хороший опыт работы с адаптивной вёрсткой, оптимизацией вёрстки для мобильных устройств.
  • less, sass, coffee script. Знаю наиболее популярные варианты семантического сахара для css и js.
  • phantom.js. Начальное знание, скрипты для перехода по страницам, тестирования страниц.
  • БЭМ, АНБ. Методология вёрстки от команды Яндекса, использовал в нескольких проектах, нахожу логичной и удобной.

Дополнительные знания

  • git и svn. В основном работал с git. Уровень создания бранчей, merge, rebase, cherry-pick.
  • grunt. Начальный уровень: создание конфигурации, сборка.
  • php и mysql. Средний уровень, могу редактировать чужой код, могу писать несложные проекты с нуля. Знания ООП имеются. С фреймворками не работал.
  • perl. Начальный уровень, писал лишь простые скрипты.
  • Firebird. Начальный уровень, доводилось работать только в рамках одного проекта.

Также знаком с большим числом cms и шаблонизаторов. Способен разобраться с любым шаблонизатором, опыт работы имеется.

Знаю английский язык на техническом уровне, читаю и пишу без словаря. В данный момент работаю над улучшением разговорного английского.
